# 🎲 Dice Roller

A simple and interactive Dice Roller application built with **HTML, CSS, and JavaScript**. The application allows users to roll one or multiple dice, generating random results and displaying matching dice images.

## ✨ Features

- Roll one or multiple dice
- Random dice generation
- Displays matching dice images
- Clean and responsive user interface
- Instant dice result display

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
